<h1 align="center">Sistema de Votación</h1>

Este proyecto es un sistema de votación que permite la creacion de mesas de votación, 
recopila todos los votos, crea dashboards y reportes con toda la información recopilada.

This is a code bundle for Sistema de Votación (Copy). The original project is available at https://www.figma.com/design/l2CwzqZyiRwN3Gygt4IhRK/Sistema-de-Votaci%C3%B3n--Copy-.

## Como Correr el Codigo

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
